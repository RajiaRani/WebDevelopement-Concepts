DataBase- Explanation

Q1 - What is database?

Ans - It is a Collection of data in a format that can be easily accessed.

Q2 - Why database ?

  1. Can store large data.
  2. features like security, scalibility.
  3. Easier to insert, delete, update and search data.

DBMS - Data Base Management System

DBMS - It is additional layer on the database.

Q3 - What is the difference between NoSql and Sql ?

SQL - It is a Rational Database.(means Data stored in the form of Tables).

Example - MySQL, Oracle, PostgreSQL etc.

NOSQL - It is a Non- Rational Database.(means Data stored in the form of Document / Key-val /graphs etc).

Example - MongoDB, Cassandra, Neo4j etc.
