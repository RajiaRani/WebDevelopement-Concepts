Active Property :

  • When using a mouse,"activation" typically starts when the user presses down the primary mouse button.
  • Targets an element that is being activated by the user, typically through clicking or tapping.

Hover Property

Applies when a user points/floats over an element with the cursor. Commonly used to change the appearance of links or buttons when moused over.

  • The :hover property, when the user interacts with an element with a pointing device, but does not necessarily activate it.
  • It is generally triggered when the user hovers over an element with the cursor (mouse pointer).

Focus Visisble :

Any-Link Property :

  • The :any-link CSS pseudo-class selector represents an element that acts as the source anchor of a hyperlink, independent of whether it has been visited. In other words, it matches every or element that has an href attribute. Thus, it matches all elements that match :link or :visited.
