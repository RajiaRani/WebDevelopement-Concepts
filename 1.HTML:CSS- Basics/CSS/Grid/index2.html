Grid Properties:

    Grid Lines

  • The lines between columns are called Column lines. The lines between rows are called Row lines.

    • Grid Cell

  • The intersection of a grid row and a grid column.

    • Grid Track

  • A grid track is the space between two adjacent grid lines .
    • A Column track is the space between two vertical grid lines, creating a column.
    • A Row track is the space between two horizontal grid lines, creating a row.

    Align Items :

      Horizontal Align

    • justify-items(Property)
      1. justify-items : start
      2. justify-items : center
      3. justify-items : end
    • justify-self(Property)

      • justify-self is only applicable for Single or Individual element.

      1. justify-self : start
      2. justify-self : center
      3. justify-self : end

      Vertical Align

    • align-items(Property)
      1. align-items : start
      2. align-items : center
      3. align-items : end

      align-self is only applicable for Single or Individual element.

    • align-self(Single Item)
      1. align-self : start
      2. align-self : center
      3. align-self : end

      If you want both horizontal & vertical property then you can use :

    • place-items : center

      • For individual items :

      1. place-self : start
      2. place-self : center
      3. place-self : end
