Grid :


Grid :

  • Setting a container's display to Grid will make all children grid items.
  • Grid is apply on the direct child of the div.
  • The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Display Property :

An HTML element becomes a grid container when its display property is set to grid or inline-grid.

  • display : grid
  • display : inline-grid
  • All direct children of the grid container automatically become grid items.

Grid Columns

  • The vertical lines of grid items are called columns.

    • Their are serval ways to write columns:

    1. grid-template-columns : auto
    2. grid-template-columns : repeat(count, 1fr)
    3. grid-template-columns : 1fr 1fr 1fr

    You can adjust the gap size by using : colmun-gap

Grid Rows

  • The horizontal lines of grid items are called rows.
  • You can adjust the gap size by using : row-gap

Grid Gaps

  • The spaces between each column/row are called gaps.

    • You can adjust the gap size by using : gap

Grid Lines

  • The lines between columns are called Column lines. The lines between rows are called Row lines.

    • Grid Cell :

  • The intersection of a grid row and a grid column.

    • Grid Track :

  • A grid track is the space between two adjacent grid lines .
    • A Column track is the space between two vertical grid lines, creating a column.

    • A Row track is the space between two horizontal grid lines, creating a row.
  • CSS for this page is :
    1. display: grid;
    2. grid-template-columns: repeat(3,1fr);
    3. grid-template-rows: repeat(2,1fr);
    4. grid-gap: 10px;
