Library

  • Library jise humara koi big code hota hai uss mein koi small task karne ke liye likhi jati hai.
  • library mein humare pass full control hota hai ki hum library ko kise aur kaha use karna chate hai.

Framework

  • A framework is a set of pre-written code that provides a structure for developing software applications.
  • Framework overall structure banne ke liye work karti hai.
  • Framework mein most of the control framework ke pass hi hota hai ki kise code likha jana chahiye.
  • example: express
