Get Request

  • Used to GET some response.
  • Data sent in query strings(limited, string data, visible in URL).
  • Hum jab bgi koi url send karte hai tho uss url ke data ki limit hoti hai.
  • Drawback : Jab bhi hum browser ko url mein koi bhi data sent karte hai to wo pura data url mein show hota hai.
